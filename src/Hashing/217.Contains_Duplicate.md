## ðŸ”¹ Code Snippet (Optimized)

```java

import java.util.HashSet;
import java.util.Set;

class Solution {
    public boolean containsDuplicate(int[] nums) {
        Set<Integer> intSet = new HashSet<>();
        for (int num : nums) {
            if (intSet.contains(num)) {
                return true;
            }
            intSet.add(num);
        }
        return false;
    }
}
```
